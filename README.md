# Installer Script

Install many tools for development using this script easily, with just one command and you are ready to go(No PATH/Environment Variables setup required).

| OS | Arch | Support |
| :---: | :---: | :---: |
| Linux | x86_64 | [x] |
| Linux | arm64 | [x] |
| Windows | x86_64 | [x] |
| Windows | arm64 | [x] |
| MacOS | x86_64 | [x] |
| MacOS | arm64 | [x] |

## Usage

```bash
chmod +x install.py
python3 install.py
```

or

```bash
chmod +x install.py
python install.py
```
